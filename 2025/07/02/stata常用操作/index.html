最近给一些同学讲stata常用操作，写了一段代码总结了我之前常用的命令，也在博客上发一下吧，如果有误可以私信或者留言。

Stata 常用操作指南

一、工作环境设置与数据准备

日志管理

  • 设置工作路径
    cd “工作目录”

  • 导入内置数据集并创建日期变量
    sysuse nlsw88, clear // 载入美国女性劳动力数据

  • 生成随机日期变量（2010-2020年间）
    generate year = floor(2010 + runiform()*(2020-2010+1))
    generate month = floor(1 + runiform()*12)
    generate day = floor(1 + runiform()*28)
    generate date_num = mdy(month, day, year)
    generate date_str = string(date_num, “%tdCCYY-NN-DD”)

  • 创建序号变量
    generate n_id = _n // 行号
    gen N_id = _N // 总样本量

  • 替换并重命名变量
    replace n_id = N_id if n_id != N_id
    drop N_id
    ren n_id N_id

  • 数值与字符串互相转换
    tostring N_id, replace // 数值→字符串
    destring N_id, replace // 字符串→数值

  • 标签值转换
    decode N_id, generate(N_id_str) // 带标签数值→字符串
    encode N_id_str, generate(N_id_num) // 字符串→带标签数值

  • 浏览与筛选数据
    browse // 交互式浏览
    list if year <= 2015 // 条件筛选
    list year if inlist(idcode, 1,2,3) // 特定ID取值
    list year if inrange(idcode, 1,3) // 范围筛选

  • 缺失值处理
    list if union == . // 查找缺失值
    misstable summarize // 缺失值统计
    drop if mi(union) // 删除缺失值

  • Winsorize 缩尾处理（需安装 winsor2）
    ssc install winsor2 // 安装命令

  • 方法1：生成新变量
    winsor2 wage, cuts(1 99) gen(wage_w)

  • 方法2：替换原变量
    winsor2 wage, replace cuts(1 99)

  • 方法3：截断处理（删除异常值）
    winsor2 wage, replace cuts(1 99) trim

count if year <= 2015 // 条件计数
describe // 数据结构
summarize // 全变量概览
tabulate year // 频数分布
tabulate year, missing // 含缺失值的频数
tabulate year, summarize(wage) // 分组统计
tabulate year, summarize(wage) means // 分组均值

    1. by前缀操作（需先排序）
      sort year
      by year: gen wage_sum = sum(wage) // 工资累计求和
    1. bysort组合命令（推荐）
      bysort year: egen total_wage = total(wage) // 工资年度总和
      bysort year: egen min_wage = min(wage) // 年度最低工资
      bysort year: egen mean_wage = mean(wage) // 年度平均工资

  • 方法1：duplicates命令
    duplicates list year // 列出重复值
    duplicates drop year, force // 强制删除重复值

  • 方法2：分组标记法（可保留特定记录）
    bysort year: gen dup = _n // 生成组内序号
    keep if dup == 1 // 保留每组第一条
    drop dup

  • 快速生成分组统计量
    collapse (count) wage_count = wage // 计数
    (sum) wage_sum = wage // 总和
    (mean) wage_mean = wage // 均值
    (min) wage_min = wage // 最小值
    (max) wage_max = wage, // 最大值
    by(year)
    save collapse_data, replace
    1. foreach：遍历变量列表
      foreach var of varlist wage year {
      summarize var' // 遍历分析 if "var’" == “wage” continue // 条件跳过
      }
    1. forvalues：遍历数字序列
      forvalues i = 1/10 {
      display "Loop: i'" if i’ == 5 continue // 跳过第5次循环
      }

  • 带步长的循环
    forvalues i = 1(2)10 {
    display “Step loop: `i’”
    }

  • 局部宏（推荐）
    local work_dir “D:/StataProject”
    use “`work_dir’/data.dta”, clear

  • 全局宏（慎用）
    global work_dir “D:/StataProject”
    use “$work_dir/data.dta”, clear

  • 循环保存/读取示例
    forvalues i = 1/10 {
    local fname “data_i'.dta" save "work_dir’/fname'", replace use "work_dir’/`fname’”, clear
    summarize
    }

    1. append：纵向追加
      use “part1.dta”, clear
      forvalues i = 2/4 {
      append using “part`i’.dta”
      }
    1. merge：横向合并

  • 准备合并数据
    preserve
    keep year
    gen set = 1
    save merge_data.dta, replace
    restore

  • 执行1:1合并
    merge 1:1 year using merge_data.dta
    keep if _merge == 3 // 保留匹配成功的记录
    drop _merge // 删除合并标记

  • 精简写法
    merge 1:1 year using merge_data.dta, keep(3) nogen

  • 生成文本日期变量
    generate date_str = string(date_num, “%tdCCYY-NN-DD”)

  • 解码分类变量
    decode industry, gen(industry_str)

  • 字符串分割
    split industry_str, parse(/) gen(industry_part)

  • 子字符串提取
    gen year_part = real(substr(date_str, 1, 4)) // 提取年份
    gen set2_part = substr(“切割文本”, 1, 3) // 注意中文字符占3字节

  • 宽→长转换
    reshape long industry_part, i(idcode) j(part_num) string
    keep if !mi(industry_part) // 删除空值

  • 长→宽转换
    reshape wide industry_part, i(idcode) j(part_num) string

log close // 关闭日志文件
